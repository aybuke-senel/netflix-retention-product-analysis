# Netflix User Behavior, Churn & Retention Analysis

## Problem
Why do users stop using a streaming platform, and how can we reduce churn?
This project analyzes user watch behavior to identify churn patterns, retention drivers, and actionable product improvements.

## Objective
The goal is not only to analyze data, but to:
- Understand user behavior
- Identify churn signals
- Segment users based on engagement
- Generate product insights
- Propose features to improve retention

## Dataset
- [Netflix Watch Log](https://www.kaggle.com/datasets/arjunajn/netflix-watch-log)

## Note
This project extends the dataset by generating additional behavioral features such as:
- Session data
- Engagement metrics
- Activity gaps
- Churn labels

## Methodology
The project follows an end-to-end product analytics workflow:
- Data Understanding & Cleaning
- Feature Engineering
- SQL Metrics (DAU, Retention, Activity)
- Exploratory Data Analysis (EDA)
- User Segmentation
- Cohort Analysis
- Churn Prediction (optional)
- Product Insights & Recommendations
